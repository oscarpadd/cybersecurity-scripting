# cybersecurity-scripting
What it does: A Python script that scans network ports to find active services like SSH.
What I learned: How network sockets work, what Port 22 is used for, and how to troubleshoot Linux server storage issues.
As well as how to move files within the Ubuntu Server
Also learned about the 3-Step Push Routine (Uploading the script to this website)
  - 1st step:(Stage the file) git add port_scan.py
  - 2nd step (commit the change): git commit -m "Whatever you worked on"
  - 3rd step (Pushing it to github): git push origin main
